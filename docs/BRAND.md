# Sprocket Brand

## Name

`Sprocket`

## Positioning

Sprocket is an AI-native repo machine.

It should feel like:

- precise
- mechanical
- dependable
- clever
- workshop-built rather than corporate-polished

It should not feel like:

- a toy
- a generic chatbot wrapper
- a soft "assistant buddy" brand
- a faceless enterprise compliance product

## Mascot

Working mascot direction:

- a mechanical lobster
- articulated claws
- rivets, brass, steel, and workshop textures
- smart and slightly menacing, not cartoon-cute

Why it works:

- claws imply grabbing, fixing, and holding systems together
- the silhouette is memorable
- it creates a distinct identity in a crowded AI tooling market

## Visual Direction

- industrial forms
- sharp silhouettes
- warm metal tones rather than generic blue gradients
- bold iconography
- character used sparingly so the product still feels serious

## Tagline Territory

Possible directions:

- Agentic repo bootstrap and self-healing maintenance
- The repo machine for coding agents
- Install the baseline. Keep the system healthy.
